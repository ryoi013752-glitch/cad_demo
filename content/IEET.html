IEET

畢業生核心能力:

  1. 具備應用數學、科學及工程知識之能力
  2. 具備設計與執行實驗，以及分析與解釋數據之能力
  3. 具備設計工程系統、元件或流程之能力
  4. 具備執行工程實務所需之技術、技能與現代工具使用能力
  5. 具備發現、分析及解決工程問題之能力
  6. 具備有效溝通與團隊合作能力
  7. 瞭解工程倫理與社會責任
  8. 具備終身學習之能力

核心能力1: 工程知識與問題解決能力

教學主軸: 機械、電控、程式與系統整合

應用在「平面五軸連桿繪圖機」

學生必須理解：

  1. 平面連桿機構學（Five-bar linkage）
  2. 運動學逆運動學（IK）
  3. 幾何約束與連桿長度配置
  4. 電機驅動（伺服或步進）
  5. 曲線繪圖軌跡生成（Bezier / spline）

實習操作：

學生需要根據機構的運動學模型推算筆尖位置，解出五軸連桿各關節角度，達成軌跡控制功能。設計過程包括工程問題定義、推導以及求解的能力。

應用在 Openduck mini:

需運用：

  1. 雙足行走步態模型基礎
  2. 3D 剛體運動學、動力學
  3. 伺服馬達扭力需求計算
  4. 穩定性分析（ZMP、CoM）

實習操作：

學生必須計算每個關節在行走或擺腿時的需求扭力，並依此選擇適合的伺服馬達。

核心能力 2: 實驗能力與數據分析能力

應用在「平面五軸連桿繪圖機」

透過實際量測：

  1. 模擬與控制前的起始條件校正
  2. 馬達轉速與筆尖移動誤差
  3. 連桿加工後的尺寸誤差
  4. 繪圖軌跡精度 vs. 理論軌跡差異
  5. 機電系統的不確定性分析

實習操作：

學生輸入圓形軌跡，量測實際繪出的圓並與理論模型比對，用 MATLAB/Python 計算誤差分布，分析造成誤差的原因（如連桿鬆動、馬達反應時間不足）。

應用在 Openduck mini:

可量測：

  1. 步態穩定性（加速度感測器）
  2. 馬達電流 vs. 荷重關係
  3. 行走速度與耗能之關係

核心能力 3：工程設計能力

應用在「平面五軸連桿繪圖機」

  1. 定義需求：成本、尺寸限制、筆尖精度
  2. 設計運算：連桿構型與尺寸設計（五桿平面機構）
  3. CAD 設計：SolidWorks/Inventor/Onshape/NX/Solvespace 繪製連桿零組件
  4. 選材：鋁擠型/PLA 印製零件
  5. 製造：3D列印、CNC、雷切
  6. 測試與修改

實際操作：

透過原型機構設計組立與伺服馬達的配置，比較動態模擬與實際繪圖的差異，並分析造成誤差或系統不穩的原因。

應用在 Openduck mini:

  1. 步態需求：步幅、穩定性
  2. 關節配置：左右腳、膝、髖等10 DOF，頭、頸等4 DOF
  3. CAD 零組件繪圖 + 運動分析
  4. 材料與結構強度分析

核心能力 4：現代工程工具使用能力

使用 CAD、機器人模擬套件、控制程式、模擬 Openduck min 行進與姿態變化。

應用在 Openduck mini:

  1. SolidWorks/Inventor/Onshape/NX/Solvespace：零組件繪圖
  2. MATLAB/Python：逆運動學、軌跡計算
  3. Arduino/STM32/Rasberry Pi：馬達控制
  4. CNC/3D 印表機：零件製造
  5. ROS（Robot Operating System）：控制架構
  6. Webots：走路動態模擬
  7. Python + OpenCV：視覺功能

實際操作：

從 Openduck min 零組件繪圖到機電系統配置以及動態行走模擬的實際操作。

核心能力 5：團隊合作與溝通能力

應用在「平面五軸連桿繪圖機」

團隊分工可能為：

  1. A：機構設計
  2. B：程式與逆運動學
  3. C：電控與配線
  4. D：軌跡設計與測試
  5. E : 協作工具選擇 (Github、Onshape)

團隊需設法整合 CAD 圖、裝配圖、控制程式、誤差分析報告。

應用在 Openduck mini:

  1. 機構子組
  2. 馬達與電源系統
  3. 控制步態演算法
  4. 外觀與封裝

核心能力 6：工程倫理與自我學習

應用在「平面五軸連桿繪圖機」

  1. 若模仿網路設計，需標註引用來源，展現倫理責任。
  2. 遇到問題，需查資料、查論文，展現自主學習。

應用在 Openduck mini:

  1. 使用鋰電池需遵守安全規範
  2. 學生需自行研究步態控制法（ZMP、倒立擺模型）

核心能力 7：當代議題與國際視野

應用在「平面五軸連桿繪圖機」

  1. 探討 CNC、自動化製造、智慧機械在工業 4.0 角色
  2. 理解國際功能性機構設計的趨勢

應用在 Openduck mini:

  1. 探討寵物與人形機器人在社會中的應用
  2. 了解目前機器人發展與應用趨勢

核心能力 8 : 具備終身學習之能力

透過課程內容，了解利用電腦軟硬體針對「平面五軸連桿繪圖機」與 Openduck min 進行輔助設計與動態模擬，仍有許多議題必須持續研究與探討，往後必須持續學習，提升自我工程能力，才能符合未來就業或創業需求。
